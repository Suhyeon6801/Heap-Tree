# Heap-Tree

### 구현 내용
- heap을 구성하고 input.txt파일(n사이즈의 heap operations)을 세 가지의 형태의 tree를 출력한다.
- 'INS #'는 '#'를 heap에 insert한다.
- ‘DEL'은 heap의 root노드을 제거한다.
- ‘EOI'을 입력받으면 input은 종료한다.
